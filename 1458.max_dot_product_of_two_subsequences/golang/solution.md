  當我們走到位置 (i, j)，面對 nums1[i-1] 和 nums2[j-1] 這兩個元素時，只有一個核心問題：

  這兩個元素要配對嗎？

  nums1 = [2, 1, -2]
                ↑ 我現在在這 (i=3)

  nums2 = [3, -6]
              ↑ 我現在在這 (j=2)

  問題：-2 和 -6 要配對嗎？

  只有兩種大方向

  方向 A：要配對 ✓

  如果決定配對 -2 和 -6，那之前的元素呢？

  nums1 = [2, 1 | -2]    ← -2 被我選了
  nums2 = [3   | -6]     ← -6 被我選了
           ↑
        之前的部分怎麼辦？

  之前的部分有兩種情況：

  情況 A1：之前沒有配對任何東西（從頭開始）
          → 結果就是 (-2) × (-6) = 12

  情況 A2：之前已經有配對了，接續下去
          → 結果就是 dp[i-1][j-1] + (-2) × (-6)
          → 也就是「之前的最佳結果」+「現在這對的乘積」

  為什麼是 dp[i-1][j-1]？

  nums1 = [2, 1, -2]
           ^^^^  ← 如果 -2 要跟 -6 配對，
                    那之前只能用 [2, 1]

  nums2 = [3, -6]
           ^     ← 如果 -6 要跟 -2 配對，
                    那之前只能用 [3]

  所以之前的最佳結果 = dp[2][1]（用 [2,1] 和 [3] 的最佳結果）

  方向 B：不配對 ✗

  如果決定不配對，那要跳過誰？

  情況 B1：跳過 nums1[i-1]（不要 -2）

          nums1 = [2, 1] ← 只用這些
          nums2 = [3, -6] ← 全部可用

          → 結果就是 dp[i-1][j] = dp[2][2]


  情況 B2：跳過 nums2[j-1]（不要 -6）

          nums1 = [2, 1, -2] ← 全部可用
          nums2 = [3]        ← 只用這些

          → 結果就是 dp[i][j-1] = dp[3][1]

  整理：4 種選擇的來源

                      要配對嗎？
                     /        \
                    是          否
                   /             \
            之前有配對嗎？      跳過誰？
             /      \           /     \
            沒有     有      跳過     跳過
             |       |      nums1    nums2
             ↓       ↓        ↓        ↓
          選擇1    選擇2    選擇3    選擇4

  | 選擇 | 意義                        | 公式                                   |
  |------|-----------------------------|----------------------------------------|
  | 1    | 配對，從頭開始              | nums1[i-1] × nums2[j-1]                |
  | 2    | 配對，接續之前              | dp[i-1][j-1] + nums1[i-1] × nums2[j-1] |
  | 3    | 不配對，跳過 nums1 當前元素 | dp[i-1][j]                             |
  | 4    | 不配對，跳過 nums2 當前元素 | dp[i][j-1]                             |

  圖示

          j-1    j
         ┌─────┬─────┐
    i-1  │  ↖  │  ↑  │
         │選擇2│選擇3│
         ├─────┼─────┤
    i    │  ←  │ now │
         │選擇4│ ??? │
         └─────┴─────┘

  dp[i][j] = max(選擇1, 選擇2, 選擇3, 選擇4)
